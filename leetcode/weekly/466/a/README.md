如果 $\textit{nums}$ 所有元素都相等，无需操作，返回 $0$。

否则，由于操作后，子数组所有元素都相等，所以直接操作整个数组，就能让所有元素相等，返回 $1$。

```py [sol-Python3]
class Solution:
    def minOperations(self, nums: List[int]) -> int:
        return int(any(x != nums[0] for x in nums))
```

```java [sol-Java]
class Solution {
    public int minOperations(int[] nums) {
        for (int x : nums) {
            if (x != nums[0]) {
                return 1;
            }
        }
        return 0;
    }
}
```

```go [sol-Go]
func minOperations(nums []int) int {
	for _, x := range nums {
		if x != nums[0] {
			return 1
		}
	}
	return 0
}
```

#### 复杂度分析

- 时间复杂度：$\mathcal{O}(n)$，其中 $n$ 是 $\textit{nums}$ 的长度。
- 空间复杂度：$\mathcal{O}(1)$。
