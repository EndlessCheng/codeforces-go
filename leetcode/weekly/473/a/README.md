不断地把 $n$ 除以 $10$（下取整）直到 $0$，例如 $123\to 12\to 1\to 0$。在这个过程中的 $n\bmod 10$，即为每个数位。

对于不为零的数位 $d$，通过 $d\cdot 10^k$ 组合起来，其中 $k$ 是当前答案的十进制长度。$k$ 从 $0$ 开始，每添加一个数位就加一。

```py [sol-Python3]
class Solution:
    def removeZeros(self, n: int) -> int:
        ans = 0
        pow10 = 1
        while n:
            n, d = divmod(n, 10)
            if d > 0:
                ans += d * pow10
                pow10 *= 10
        return ans
```
