顺次数的十进制表示是 $\texttt{123456789}$ 的**子串**。

枚举 $\texttt{123456789}$ 的子串，如果其数值 $x$ 在 $[\textit{low},\textit{high}]$ 中，则把 $x$ 加入答案。

## 写法一：枚举左右端点

枚举顺次数的最高位 $d$，然后依次把 $d+1,d+2,\ldots,9$ 加到顺次数的末尾，得到新的顺次数。

例如 $d=5$，我们会依次得到顺次数 $5\to 56\to 567\to 5678\to 56789$。

注意题目要求答案是有序的，最后需要对答案排序。

> 注：本题保证顺次数长度至少是 $2$，也可以初始化顺次数为 $10d+(d+1) = 11d+1$。

```py [sol-Python3]
class Solution:
    def sequentialDigits(self, low: int, high: int) -> List[int]:
        ans = []
        for d in range(1, 10):
            x = d
            for i in range(d, 10):
                if x > high:
                    break
                if x >= low:
                    ans.append(x)
                # 把 i+1 加到 x 的末尾
                x = x * 10 + i + 1
        ans.sort()
        return ans
```

#### 复杂度分析

- 时间复杂度：$\mathcal{O}(\log^2 \textit{high}\log\log \textit{high})$。瓶颈在排序上。顺次数的十进制长度不超过 $\mathcal{O}(\log \textit{high})$，在这个长度内的子串个数为 $\mathcal{O}(\log^2 \textit{high})$。排序需要 $\mathcal{O}(\log^2 \textit{high}\log\log \textit{high})$ 的时间。
- 空间复杂度：$\mathcal{O}(1)$。忽略排序的栈开销。返回值不计入。

## 写法二：先枚举长度，再枚举位置

先枚举子串长度，再枚举子串位置，这样无需对答案排序。

```py [sol-Python3]
DIGITS = "123456789"

class Solution:
    def sequentialDigits(self, low: int, high: int) -> List[int]:
        ans = []
        # 枚举 DIGITS 的子串（先枚举子串长度，再枚举子串位置）
        for length in range(len(str(low)), len(str(high)) + 1):
            for r in range(length, len(DIGITS) + 1):
                x = int(DIGITS[r - length: r])
                if low <= x <= high:
                    ans.append(x)
        return ans
```

```java [sol-Java]
class Solution {
    private static final String digits = "123456789";

    public List<Integer> sequentialDigits(int low, int high) {
        List<Integer> ans = new ArrayList<>();
        int minLen = String.valueOf(low).length();
        int maxLen = String.valueOf(high).length();
        // 枚举 digits 的子串（先枚举子串长度，再枚举子串位置）
        for (int length = minLen; length <= maxLen; length++) {
            for (int r = length; r <= digits.length(); r++) {
                int x = Integer.parseInt(digits.substring(r - length, r));
                if (low <= x && x <= high) {
                    ans.add(x);
                }
            }
        }
        return ans;
    }
}
```

                }
            }
        }
        return ans;
    }

	// 枚举 digits 的子串（先枚举子串长度，再枚举子串位置）

				ans = append(ans, x)
			}
		}
	}
	return

#### 复杂度分析

- 时间复杂度：$\mathcal{O}(\log^3 \textit{high})$。枚举了 $\mathcal{O}(\log^2 \textit{high})$ 个子串，处理每个子串需要 $\mathcal{O}(\log \textit{high})$ 的时间。
- 空间复杂度：$\mathcal{O}(\log \textit{high})$。返回值不计入。

## 写法三（优化）：定长滑动窗口

用定长滑动窗口，我们可以 $\mathcal{O}(1)$ 计算下一个子串的数值。

```py [sol-Python3]
class Solution:
    def sequentialDigits(self, low: int, high: int) -> List[int]:
        ans = []
        x0 = 12  # 第一个窗口
        pow10 = 10
        length = 2  # 窗口长度，同时也是第一个窗口的个位数字
        while x0 <= high:
            pow10 *= 10
            x = x0
            for i in range(length, 10):
                if x > high:
                    return ans
                if x >= low:
                    ans.append(x)
                # 窗口向右滑动，i+1 进入窗口，i+1-length 离开窗口
                x = x * 10 + i + 1 - (i + 1 - length) * pow10
            length += 1
            x0 = x0 * 10 + length
        return ans
```
