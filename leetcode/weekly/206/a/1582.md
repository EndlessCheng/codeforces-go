对于矩阵的每一行，如果这一行只有一个 $1$，那么遍历 $1$ 所在的那一列，如果也只有一个 $1$，则把答案加一。

```py [sol-Python3]
class Solution:
    def numSpecial(self, mat: List[List[int]]) -> int:
        ans = 0
        for row in mat:
            if sum(row) != 1:
                continue
            j = row.index(1)
            # 计算 j 列的元素和，必须恰好是 1
            if sum(row[j] for row in mat) == 1:
                ans += 1
        return ans
```
