回想一下力扣第一题的 [哈希表做法](https://leetcode.cn/problems/two-sum/solution/dong-hua-cong-liang-shu-zhi-he-zhong-wo-0yvmj/)。

不妨设 $i<j$。枚举 $\textit{nums}[j]$，设 $\textit{nums}[j]$ 的数位和为 $s$，我们需要知道 $j$ 左边的数位和也等于 $s$ 的**最大**的 $\textit{nums}[i]$。可以用一个哈希表（或者数组）$\textit{mx}[s]$ 记录数位和为 $s$ 对应的最大的 $\textit{nums}[i]$。

用 $\textit{mx}[s]+\textit{nums}[j]$ 更新答案的最大值。 

如果不存在两个数位和相同的数，返回 $-1$。

```py [sol-Python3]
class Solution:
    def maximumSum(self, nums: List[int]) -> int:
        ans = -1
        mx = [-inf] * 82  # 至多 9 个 9 相加
        for num in nums:  # 枚举 num = nums[j]
            s = sum(map(int, str(num)))  # 更快的写法见【Python3 写法二】
            ans = max(ans, mx[s] + num)  # 左边找一个数位和也为 s 的最大的 nums[i]
            mx[s] = max(mx[s], num)  # 维护数位和等于 s 的最大元素
        return ans
```

```py [sol-Python3 写法二]
class Solution:
    def maximumSum(self, nums: List[int]) -> int:
        ans = -1
        mx = [-inf] * 82  # 至多 9 个 9 相加
        for num in nums:  # 枚举 num = nums[j]
            s = 0
            x = num
            while x:  # 枚举 num 的每个数位
                s += x % 10
                x //= 10
            ans = max(ans, mx[s] + num)  # 左边找一个数位和也为 s 的最大的 nums[i]
            mx[s] = max(mx[s], num)  # 维护数位和等于 s 的最大元素
        return ans
```

```java [sol-Java]
class Solution {

        int ans = -1;
        int[] mx = new int[82]; // 至多 9 个 9 相加
        Arrays.fill(mx, Integer.MIN_VALUE);

        }
        return ans;
    }
}
```

```rust [sol-Rust]
impl Solution {
    pub fn maximum_sum(nums: Vec<i32>) -> i32 {
        let mut ans = -1;
        let mut mx = [i32::MIN; 82]; // 至多 9 个 9 相加
        for num in nums { // 枚举 num = nums[j]
            let mut s = 0; // num 的数位和
            let mut x = num;
            while x > 0 { // 枚举 num 的每个数位
                s += (x % 10) as usize;
                x /= 10;
            }
            ans = ans.max(mx[s] + num); // 左边找一个数位和也为 s 的最大的 nums[i]
            mx[s] = mx[s].max(num); // 维护数位和等于 s 的最大元素
        }
        ans
    }
}
```

#### 复杂度分析

- 时间复杂度：$\mathcal{O}(n\log U)$，其中 $n$ 为 $\textit{nums}$ 的长度，$U=\max(\textit{nums})$。
- 空间复杂度：$\mathcal{O}(D\log U)$，其中 $D=9$。
