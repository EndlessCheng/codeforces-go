本题有两种写法：

1. 使用库函数，把 $\textit{nums}$ 拼到 $\textit{nums}$ 的末尾。
2. 创建一个长为 $2n$ 的数组 $\textit{ans}$，手动把元素一个一个地填入 $\textit{ans}$。

## 写法一：库函数

```py [sol-Python3]
class Solution:
    def getConcatenation(self, nums: List[int]) -> List[int]:
        return nums * 2  # 或者 nums + nums，或者 nums += nums 然后返回 nums
```

```java [sol-Java]
class Solution {
    public int[] getConcatenation(int[] nums) {
        int n = nums.length;
        int[] ans = new int[n * 2];
        System.arraycopy(nums, 0, ans, 0, n);
        System.arraycopy(nums, 0, ans, n, n);
        return ans;
    }
}
```

```cpp [sol-C++]
class Solution {
public:
    vector<int> getConcatenation(vector<int>& nums) {
        nums.insert(nums.end(), nums.begin(), nums.end());
        return nums;
    }
};
```

```c [sol-C]
int* getConcatenation(int* nums, int numsSize, int* returnSize) {
    *returnSize = numsSize * 2;
    int* ans = malloc(numsSize * 2 * sizeof(int));
    memcpy(ans, nums, numsSize * sizeof(int));
    memcpy(ans + numsSize, nums, numsSize * sizeof(int));
    return ans;
}
```

```go [sol-Go]
func getConcatenation(nums []int) []int {
	return append(nums, nums...)
}
```

```js [sol-JavaScript]
var getConcatenation = function(nums) {
    return nums.concat(nums);
};
```

```rust [sol-Rust]
impl Solution {
    pub fn get_concatenation(nums: Vec<i32>) -> Vec<i32> {
        [nums.clone(), nums].concat()
    }
}
```

```rust [sol-Rust 写法二]
impl Solution {
    pub fn get_concatenation(mut nums: Vec<i32>) -> Vec<i32> {
        nums.extend(nums.clone());
        nums
    }
}
```

#### 复杂度分析

- 时间复杂度：$\mathcal{O}(n)$，其中 $n$ 是 $\textit{nums}$ 的长度。
- 空间复杂度：$\mathcal{O}(1)$。返回值不计入。

## 写法二：手动复制元素

```py [sol-Python3]
class Solution:
    def getConcatenation(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = [0] * (n * 2)
        for i, x in enumerate(nums):
            ans[i] = ans[i + n] = x
        return ans
```

```java [sol-Java]
class Solution {
    public int[] getConcatenation(int[] nums) {
        int n = nums.length;
        int[] ans = new int[n * 2];
        for (int i = 0; i < n; i++) {
            ans[i] = ans[i + n] = nums[i];
        }
        return ans;
    }
}
```

func getConcatenation(nums []int) []int {
	n := len(nums)
	ans := make([]int, n*2)

```rust [sol-Rust]
impl Solution {
    pub fn get_concatenation(nums: Vec<i32>) -> Vec<i32> {
        let n = nums.len();
        let mut ans = vec![0; n * 2];
        for (i, x) in nums.into_iter().enumerate() {
            ans[i] = x;
            ans[i + n] = x;
        }
        ans
    }
}
```

- 时间复杂度：$\mathcal{O}(n)$，其中 $n$ 是 $\textit{nums}$ 的长度。
- 空间复杂度：$\mathcal{O}(1)$。返回值不计入。
