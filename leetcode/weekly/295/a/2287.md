统计 $s$ 每个字母的出现次数 $\textit{cntS}$，统计 $\textit{target}$ 每个字母的出现次数 $\textit{cntT}$。

例如 $s$ 中有 $7$ 个 $\texttt{a}$，一个 $\textit{target}$ 需要 $2$ 个 $\texttt{a}$，那么至多组成 $\left\lfloor\dfrac{7}{2}\right\rfloor=3$ 个 $\textit{target}$。

一般地，枚举 $\textit{cntT}[i]>0$ 的字母 $i$，计算 $\left\lfloor\dfrac{\textit{cntS}[i]}{\textit{cntT}[i]}\right\rfloor$ 的最小值，即为答案。

```py [sol-Python3]
class Solution:
    def rearrangeCharacters(self, s: str, target: str) -> int:
        cnt_s = Counter(s)
        cnt_t = Counter(target)
        return min(cnt_s[ch] // cnt for ch, cnt in cnt_t.items())
```

```java [sol-Java]
class Solution {
    public int rearrangeCharacters(String s, String target) {
        int[] cntS = new int[26];
        for (char ch : s.toCharArray()) {
            cntS[ch - 'a']++;
        }

        int[] cntT = new int[26];
        for (char ch : target.toCharArray()) {
            cntT[ch - 'a']++;
        }

        int ans = Integer.MAX_VALUE;
        for (int i = 0; i < 26; i++) {
            if (cntT[i] > 0) {
                ans = Math.min(ans, cntS[i] / cntT[i]);
            }
        }
        return ans;
    }
}
```

```go [sol-Go]
func rearrangeCharacters(s, target string) int {
	var cntS, cntT [26]int
	for _, ch := range s {
		cntS[ch-'a']++
	}
	for _, ch := range target {
		cntT[ch-'a']++
	}

	ans := math.MaxInt
	for i, ct := range cntT {
		if ct > 0 {
			ans = min(ans, cntS[i]/ct)
		}
	}
	return ans
}
```

#### 复杂度分析

- 时间复杂度：$\mathcal{O}(n+m)$ 或 $\mathcal{O}(n + m + |\Sigma|)$，其中 $n$ 是 $s$ 的长度，$m$ 是 $\textit{target}$ 的长度，$|\Sigma|=26$ 是字符集合的大小。
- 空间复杂度：$\mathcal{O}(|\Sigma|)$。
