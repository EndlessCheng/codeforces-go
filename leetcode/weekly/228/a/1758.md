交替字符串只有两种：$\texttt{010101}\cdots$ 和 $\texttt{101010}\cdots$ 分别计算对应的修改次数。

比如 $s = \texttt{0100}$，变成 $\texttt{0101}$，只有 $s[3]$ 不一样，修改 $1$ 次；变成 $\texttt{1010}$，$s[0],s[1],s[2]$ 不一样，修改 $3$ 次。

一般地，如果变成 $t_0 = \texttt{010101}\cdots$ 需要修改 $k$ 次，那么由于 $t_1 = \texttt{101010}\cdots$ 每个位置都和 $t_0$ 不同，变成 $t_0$ 需要修改的字符，变成 $t_1$ 无需修改；变成 $t_0$ 不需要修改的字符，变成 $t_1$ 需要修改。所以变成 $t_1$ 需要修改 $n-k$ 次。答案为 $\min(k, n-k)$。

```py [sol-Python3]
class Solution:
    def minOperations(self, s: str) -> int:
        diff = 0
        for i, ch in enumerate(s):
            # 如果 i 是偶数，把 ch 变成 0，否则变成 1
            if int(ch) != i % 2:
                diff += 1
        return min(diff, len(s) - diff)
```

#### 复杂度分析

- 时间复杂度：$\mathcal{O}(n)$，其中 $n$ 是 $s$ 的长度。
- 空间复杂度：$\mathcal{O}(1)$。
