根据题意，如果 $\textit{nums}$ 是由一个递增数组旋转得到的，那么 $\textit{nums}$ 至多有两个递增段。换句话说，$\textit{nums}$ **至多有一对相邻元素是严格递减的**，即 $\textit{nums}[i-1] > \textit{nums}[i]$ 至多出现一次。此外，如果有两个递增段，第二段的最大值不能超过第一段的最小值，即 $\textit{nums}[0]\ge \textit{nums}[n-1]$。

## 写法一

```py [sol-Python3]
class Solution:
    def check(self, nums: list[int]) -> bool:
        is_sorted = True
        for x, y in pairwise(nums):
            if x > y:  # 严格递减
                if not is_sorted:  # 之前出现过严格递减，说明至少有三个递增段
                    return False
                is_sorted = False  # 标记遇到了严格递减
        return is_sorted or nums[0] >= nums[-1]
```

```java [sol-Java]

    public boolean check(int[] nums) {
        int n = nums.length;
        boolean sorted = true;
        for (int i = 1; i < n; i++) {
            if (nums[i - 1] > nums[i]) { // 严格递减

                    return false;
                }
                sorted = false; // 标记遇到了严格递减
            }
        }
        return sorted || nums[0] >= nums[n - 1];
    }
}
```
