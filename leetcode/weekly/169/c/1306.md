用图论术语描述，问题等价于：

- 给你一个 $n$ 个节点的有向图，节点编号从 $0$ 到 $n-1$。图中的边为 $i\to i+\textit{arr}[i]$ 和 $i\to i-\textit{arr}[i]$（如果节点存在）。
- 判断能否从起点 $\textit{start}$ 出发，到达任意一个满足 $\textit{arr}[i]=0$ 的节点 $i$。

从 $\textit{start}$ 开始，DFS（或者 BFS）访问这张图。找到 $\textit{arr}[i]=0$ 的节点 $i$ 时，返回 $\texttt{true}$。

```py [sol-Python3]
class Solution:
    def canReach(self, arr: list[int], start: int) -> bool:
        n = len(arr)
        vis = [False] * n

        def dfs(i: int) -> bool:
            if not 0 <= i < n or vis[i]:  # 出界，或者之前访问过（没找到）
                return False
            if arr[i] == 0:  # 找到了
                return True
            vis[i] = True  # 避免重复访问
            return dfs(i + arr[i]) or dfs(i - arr[i])

        return dfs(start)
```

```java [sol-Java]
class Solution {
    public boolean canReach(int[] arr, int start) {
        int n = arr.length;
        boolean[] vis = new boolean[n];
        return dfs(start, arr, vis);
    }

    private boolean dfs(int i, int[] arr, boolean[] vis) {
        if (i < 0 || i >= vis.length || vis[i]) { // 出界，或者之前访问过（没找到）
            return false;
        }
        if (arr[i] == 0) { // 找到了
            return true;
        }
        vis[i] = true; // 避免重复访问
        return dfs(i + arr[i], arr, vis) || dfs(i - arr[i], arr, vis);
    }
}
```

```rust [sol-Rust]
impl Solution {
    pub fn can_reach(arr: Vec<i32>, start: i32) -> bool {
        fn dfs(i: usize, arr: &[i32], vis: &mut [bool]) -> bool {
            if i >= vis.len() || vis[i] { // 出界，或者之前访问过（没找到）
                return false;
            }
            if arr[i] == 0 { // 找到了
                return true;
            }
            vis[i] = true; // 避免重复访问
            let d = arr[i] as usize;
            dfs(i + d, arr, vis) || dfs(i - d, arr, vis)
        }

        let mut vis = vec![false; arr.len()];
        dfs(start as usize, &arr, &mut vis)
    }
}
```

#### 复杂度分析

- 时间复杂度：$\mathcal{O}(n)$，其中 $n$ 是 $\textit{arr}$ 的长度。图有 $\mathcal{O}(n)$ 个点，$\mathcal{O}(n)$ 条边，DFS 需要 $\mathcal{O}(n)$ 的时间。
- 空间复杂度：$\mathcal{O}(n)$。
