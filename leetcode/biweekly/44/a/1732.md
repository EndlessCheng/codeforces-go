设 $h$ 为海拔高度数组。根据题意，我们有

- $h[0] = 0$。
- $h[1] - h[0] = \textit{gain}[0]$。
- $h[2] - h[1] = \textit{gain}[1]$。
- ……
- $h[i] - h[i-1] = \textit{gain}[i-1]$。

左边累加得

$$
(h[i]-h[i-1]) + (h[i-1]-h[i-2]) + \cdots + (h[1] - h[0]) + h[0] = h[i]
$$

右边累加得

$$
\textit{gain}[0] + \cdots + \textit{gain}[i-2] + \textit{gain}[i-1]
$$

所以 $h$ 是 $\textit{gain}$ 的前缀和数组。

> 本质上，$\textit{gain}$ 是 $h$ 的差分数组。计算差分数组的前缀和，就得到了 $h$。

最后返回 $\max(h)$。

```py [sol-Python3]
class Solution:
    def largestAltitude(self, gain: List[int]) -> int:
        return max(accumulate(gain, initial=0))
```
