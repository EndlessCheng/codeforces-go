一圈是 $360^\circ$，有 $60$ 个刻度。每过 $1$ 分钟，分针顺时针转了 $\dfrac{360^\circ}{60} = 6^\circ$。每过 $1$ 小时，时针顺时针转了 $\dfrac{360^\circ}{12} = 30^\circ$。

分针从 $0$ 开始，顺时针转到 $\textit{minutes}$，转了 $6\cdot \textit{minutes}^\circ$。

时针从 $0$ 开始，顺时针转到 $\textit{hour}$ 时 $\textit{minutes}$ 分，即 $\textit{hour} + \dfrac{\textit{minutes}}{60}$ 时，转了 $30\cdot \left(\textit{hour} + \dfrac{\textit{minutes}}{60}\right)^\circ$。

所以时针和分针的夹角为

$$
d = {\left|30\cdot \left(\textit{hour} + \dfrac{\textit{minutes}}{60}\right) - 6\cdot \textit{minutes}\right|}^\circ = {|30\times \textit{hour} - 5.5\times\textit{minutes}|}^\circ
$$

注意 $d$ 可能大于 $180^\circ$。题目要求较小角，即

$$
\min(d, 360^\circ - d)
$$

**注**：根据计算 $d$ 的式子，在本题数据范围下，$d\le 360^\circ$，我们不需要把 $\textit{hour}$ 更新为 $\textit{hour}\bmod 12$ 再计算。如果 $\textit{hour} \ge 13$，则需要取模。

## 写法一

```py [sol-Python3]
class Solution:
    def angleClock(self, hour: int, minutes: int) -> float:
        d = abs(hour * 30 - minutes * 5.5)
        return min(d, 360 - d)
```

```java [sol-Java]
class Solution {
    public double angleClock(int hour, int minutes) {
        double d = Math.abs(hour * 30 - minutes * 5.5);
        return Math.min(d, 360 - d);
    }
}
```

## 写法二：尽量避免浮点运算

先计算

$$
2d = {|60\times \textit{hour} - 11\times\textit{minutes}|}^\circ
$$

以及

$$
\min(2d, 720^\circ - 2d)
$$

最后再除以 $2$，这样只会执行一次浮点运算。

```py [sol-Python3]
class Solution:
    def angleClock(self, hour: int, minutes: int) -> float:
        d = abs(hour * 60 - minutes * 11)
        return min(d, 720 - d) / 2
```

```java [sol-Java]
class Solution {
    public double angleClock(int hour, int minutes) {
        int d = Math.abs(hour * 60 - minutes * 11);
        return Math.min(d, 720 - d) / 2.0;
    }
}
```

```cpp [sol-C++]

public:

```

}

func abs(x int) int {

		return -x
	}

}
