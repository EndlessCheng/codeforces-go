## 方法一：暴力枚举

先说暴力怎么做。

我们可以枚举所有 $a>b$ 的平方和三元组 $(a,b,c)$，由于 $a^2+b^2=b^2+a^2$，所以 $(b,a,c)$ 也是平方和三元组，所以只需考虑 $a>b$ 的情况。最后把统计结果乘以 $2$，即为答案。

你可能会问，$a=b$ 的情况呢？由于 $2a^2 = c^2$ 成立时，$c = \sqrt 2 a$，所以 $c$ 不是整数（式子变形得 $\dfrac{c}{a} = \sqrt 2$，左边是有理数，右边是无理数，有理数不可能等于无理数）。所以无需考虑 $a=b$ 的情况。

枚举 $a=2,3,\ldots,n-1$，枚举 $b=1,2,\ldots,a-1$，如果 $a^2+b^2\le n$ 且 $c = \sqrt{a^2+b^2}$ 是整数，那么我们找到了一个平方和三元组 $(a,b,c)$，计数器加一。

```py [sol-Python3]
class Solution:
    def countTriples(self, n: int) -> int:
        ans = 0
        for a in range(1, n):
            for b in range(1, a):
                if a * a + b * b > n * n:
                    break
                c2 = a * a + b * b
                if isqrt(c2) ** 2 == c2:
                    ans += 1
        return ans * 2  # (a,b,c) 和 (b,a,c) 各算一次
```

## 方法二：本原勾股数组

请看 [勾三股四弦五：生成勾股数的公式](https://zhuanlan.zhihu.com/p/1978057341963368012)。

```py [sol-Python3]
class Solution:
    def countTriples(self, n: int) -> int:
        ans = 0
        u = 3
        while u * u < n * 2:
            v = 1
            while v < u and (u * u + v * v) // 2 <= n:
                if gcd(u, v) == 1:
                    c0 = (u * u + v * v) // 2
                    ans += n // c0
                v += 2
            u += 2
        return ans * 2  #  (a,b,c) 和 (b,a,c) 各算一次
```

```java [sol-Java]
class Solution {
    public int countTriples(int n) {
        int ans = 0;
        for (int u = 3; u * u < n * 2; u += 2) {
            for (int v = 1; v < u && (u * u + v * v) / 2 <= n; v += 2) {
                if (gcd(u, v) == 1) {
                    int c0 = (u * u + v * v) / 2;
                    ans += n / c0;
                }
            }
        }
        return ans * 2; //  (a,b,c) 和 (b,a,c) 各算一次
    }

    private int gcd(int a, int b) {
        while (a != 0) {
            int tmp = a;
            a = b % a;
            b = tmp;
        }
        return b;
    }
}
```

```rust [sol-Rust]
impl Solution {
    pub fn count_triples(n: i32) -> i32 {
        let mut ans = 0;
        let mut u = 3;
        while u * u < n * 2 {
            let mut v = 1;
            while v < u && (u * u + v * v) / 2 <= n {
                if gcd(u, v) == 1 {
                    let c0 = (u * u + v * v) / 2;
                    ans += n / c0;
                }
                v += 2;
            }
            u += 2;
        }

    while a != 0 {
        (a, b) = (b % a, a);
    }
    b
```

#### 复杂度分析

- 时间复杂度：$\mathcal{O}(n\log n)$ 或 $\mathcal{O}(n)$。枚举 $\mathcal{O}(\sqrt n\cdot \sqrt n) = \mathcal{O}(n)$ 次，每次花费 $\mathcal{O}(\log n)$ 的时间计算 GCD。如果预处理 GCD，则可以做到 $\mathcal{O}(n)$ 时间，参考 Python3 写法二。
- 空间复杂度：$\mathcal{O}(1)$。
