首先计算有多少个长为 $n$ 的开心字符串。 

这是一个计数问题。有 $n$ 个空位，第一个位置可以填 $3$ 种字母，后面的每个位置，都不能和前一个位置的字母相同，所以都只有 $2$ 种填法，因此方案数为

$$
3\cdot 2^{n-1}
$$

如果 $k > 3\cdot 2^{n-1}$，返回空串。否则答案是存在的。

---

为方便计算，我们先把 $k$ **减一**，改成从 $0$ 开始。

以 $n=4$，$k=12=1100_{(2)}$（**减一后**）为例说明。

答案的第一个字母是什么？

如果第一个字母是 $\texttt{a}$，那么后面的 $n-1=3$ 个位置有 $2^{n-1}=8$ 种填法，不够。

如果第一个字母是 $\texttt{b}$，同样地，后面的 $n-1=3$ 个位置有 $2^{n-1}=8$ 种填法，这就够了，所以答案的第一个字母填 $\texttt{b}$。

现在答案为 $\texttt{b\_\_\_}$，由于剩余的每个位置都有两种填法，这和 $k$ 二进制的低三位 $100$ 完美对应：

- 答案的第二个字母填 $\texttt{a}$ 或者 $\texttt{c}$，由于 $k$ 这一位是 $1$，所以填 $\texttt{c}$。
- 答案的第三个字母填 $\texttt{a}$ 或者 $\texttt{b}$，由于 $k$ 这一位是 $0$，所以填 $\texttt{a}$。
- 答案的第四个字母填 $\texttt{b}$ 或者 $\texttt{c}$，由于 $k$ 这一位是 $0$，所以填 $\texttt{b}$。

所以答案为 $\texttt{bcab}$。

一般地，答案的第一个字母是第 $\left\lfloor\dfrac{k}{2^{n-1}}\right\rfloor$ 个小写字母，随后的字母可以根据 $k\bmod 2^{n-1}$ 二进制从高到低是 $0$ 还是 $1$，填入相应的字母：

- 首先，如果二进制这一位是 $0$，那么先填入 $\texttt{a}$，否则填入 $\texttt{b}$。
- 然后看填入的字母是否大于等于左侧相邻字母，如果大于等于，那么把填入的字母加一。比如左侧相邻字母是 $\texttt{a}$，那么如果我们填的是 $\texttt{a}$，变成 $\texttt{b}$；如果填的是 $\texttt{b}$，变成 $\texttt{c}$。

> **注**：这个「加一」的技巧可用于生成两个不同的随机整数，见 [961. 在长度 2N 的数组中找出重复 N 次的元素](https://leetcode.cn/problems/n-repeated-element-in-size-2n-array/) 我的题解的方法四。

```py [sol-Python3]
class Solution:
    def getHappyString(self, n: int, k: int) -> str:
        if k > 3 << (n - 1):
            return ""
        k -= 1  # 改成从 0 开始，方便计算
        ans = [ord('a')] * n
        ans[0] += k >> (n - 1)
        for i in range(1, n):
            ans[i] += k >> (n - 1 - i) & 1
            if ans[i] >= ans[i - 1]:
                ans[i] += 1
        return ''.join(map(chr, ans))
```

impl Solution {
    pub fn get_happy_string(n: i32, mut k: i32) -> String {
        if k > 3 << (n - 1) {
            return String::new();
        }
        k -= 1; // 改成从 0 开始，方便计算
        let n = n as usize;
        let mut ans = vec![0; n];
        ans[0] = b'a' + (k >> (n - 1)) as u8;
        for i in 1..n {
            ans[i] = b'a' + (k >> (n - 1 - i) & 1) as u8;
            if ans[i] >= ans[i - 1] {
                ans[i] += 1;
            }
        }
        unsafe { String::from_utf8_unchecked(ans) }
    }
}
