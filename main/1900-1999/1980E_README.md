点评：结论好猜，重点在证明上。

操作不会改变什么？

列与列的交换不会改变每一行的元素（元素集合不变）。
行与行的交换不会改变每一列的元素（元素集合不变）。

所以矩阵 a 要能变成矩阵 b，必要条件是两个矩阵有相同的行集合，有相同的列集合。
意思是 (a 的每一行排序，整个 a 再排序) == (b 的每一行排序，整个 b 再排序)。列也同理。

下面说明这也是充分的。意思是，如果操作后，a 的每一行的元素集合与 b 相等，每一列的元素集合与 b 相等，那么 a == b 一定成立。
你可能会问：我们只保证了集合相等，万一元素顺序不等呢？

比如交换后，a 和 b 第一行的元素集合都是 {1,5,9}，第三列的元素集合都是 {1,3,6}。
这两个集合的交集是 {1}，意味着什么？
意味着元素 1 在两个矩阵中的位置是唯一确定的，都位于第一行第三列。即操作后，a[1][3] == b[1][3]。

一般地，在矩阵元素互不相同的前提下，恰好有一个数既属于第 i 行，又属于第 j 列。
如果操作后，a 和 b 的第 i 行的元素集合相等，第 j 列的元素集合相等，那么行列元素集合的交集也相等。在矩阵元素互不相同的前提下，交集的大小恰好是 1。
交集元素即第 i 行第 j 列的元素，这意味着操作后，a[i][j] == b[i][j]，所以 a == b。
